# storyly-placement-react-native

React-native package for Storyly Placement

## Installation


```sh
npm install storyly-placement-react-native
```


## Usage


```js
import { StorylyPlacementReactNativeView } from "storyly-placement-react-native";

// ...

<StorylyPlacementReactNativeView color="tomato" />
```
